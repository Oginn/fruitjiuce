# 🍓 Fruitjuice Website Project

## Project Overview

This project is a simple, responsive "Fruitjuice" website designed to teach students the fundamentals of web development using **HTML**, **CSS**, and **JavaScript**. It serves as a practical, hands-on example for understanding basic website structure, styling, and interactivity.

## Motivation (Why this project?)

The motivation behind this project was to provide a real-world, yet simple, case study for students to:

*   **Understand basic file structure** of a website (HTML, CSS, JS folders/files).
*   **Practice semantic HTML5** elements.
*   **Learn CSS fundamentals** including layout (Flexbox/Grid), styling, and responsiveness.
*   **Explore basic JavaScript** for simple interactivity (e.g., a simple menu toggle or an alert).
*   **Build a complete, viewable project** that demonstrates acquired skills to potential employers or for personal portfolios.

## Features

*   Simple, clean user interface.
*   Responsive design, adaptable to various devices (mobile, tablet, desktop).
*   Basic navigation menu.
*   "Juice of the Day" interactive section (using JavaScript).
*   Clear separation of concerns (HTML for structure, CSS for presentation, JS for behavior).

## How to View the Project

This project is a static website and does not require any complex installation or backend setup.

*   **Live Demo:** If hosted online, add the link here: `(https://oginn.github.io/fruitjiuce/)`.
*   **Locally:**
    1.  **Clone the repository** to your local machine:
        ```bash
         https://github.com/Oginn/
        ```
    2.  **Navigate** to the project directory:
        ```bash
        cd fruitjuice-website
        ```
    3.  **Open the `index.html` file** in your preferred web browser.

## Technologies Used

*   **HTML5:** For the core structure of the website.
*   **CSS3:** For all styling and layout (e.g., colors, fonts, responsiveness).
*   **JavaScript:** For any interactive elements (e.g., menu toggle).
